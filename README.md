## 项目简介

本项目是一个系统化的英语语法学习资源库，通过结构化的大纲、专业的提示词模板和高质量的练习题，帮助学习者从高中英语水平逐步提升到考研英语水平。

### 核心特点

- **系统化学习**：从词法到语法，从基础到进阶，循序渐进
- **难度分级**：高中基础 → 考研铺垫 → 考研提升
- **练习导向**：每章包含综合练习题和口语练习
- **AI 辅助**：使用提示词模板自动生成高质量练习题
- **完整覆盖**：涵盖词法、语法、综合应用三大模块

## 快速开始

### 1. 查看学习大纲

打开 [outline.md](outline.md) 了解完整的学习内容和结构。

### 2. 开始学习

选择一个章节开始学习，例如从 [第一章：名词](src/part1/chapter1/noun_exercises.md) 开始。

### 3. 完成练习

每个章节包含：
- **知识点概览**：核心定义、分类、关键用法、易错点
- **综合练习题**（12-15题）：单选、改错、翻译
- **口语综合练习**（4个任务）：跟读、情景描述
- **高频易错点汇总**：常见错误及正确用法
- **学习建议**：实用学习技巧

## 使用指南

### 对于学习者

1. **按顺序学习**：建议按照大纲顺序逐章学习
2. **完成练习**：每章完成后完成所有练习题
3. **口语练习**：不要忽略口语练习，开口练习很重要
4. **复习易错点**：重点关注高频易错点汇总
5. **循序渐进**：先掌握高中基础，再过渡到考研难度

## 练习题示例

以下是第一章（名词）的练习题示例：

### 单项选择题

**1. There are many ______ in the library.**
A. book
B. books
C. bookes
D. book's

> **答案：B**
> **解析：** 考查可数名词复数。book是规则可数名词，复数直接加-s，books。

### 口语练习

**任务一：核心句式跟读**
- I have three books.
- This is my friend's car.
- There is some water in the bottle.

更多完整示例请查看 [noun_exercises.md](src/part1/chapter1/noun_exercises.md)。

## 学习建议

1. **打好基础**：先掌握词法基础，再学习语法规则
2. **多练多记**：完成练习题后，定期复习易错点
3. **开口练习**：不要只做笔试，口语练习同样重要
4. **循序渐进**：不要急于求成，按照大纲顺序学习
5. **学以致用**：将学到的语法知识应用到实际阅读和写作中
