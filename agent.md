# English Grammar Book Agent

## 项目概述

本项目是一个系统化的英语语法学习项目，从高中英语水平起步，逐步过渡到考研英语水平。通过结构化的大纲、专业的提示词模板和自动生成的练习题，帮助学习者系统掌握英语语法知识。

## 目录结构

```
grammar-book/
├── outline.md                    # 学习大纲（词法+语法+综合应用）
├── prompt/                       # 提示词模板目录
│   ├── template.md              # 考研导向模板
│   └── template_mid.md          # 高中基础模板（当前使用）
├── src/                         # 生成的练习题目录
│   └── part1/                   # 第一部分：词法
│       └── chapter1/            # 第一章：名词
│           └── noun_exercises.md
└── agent.md                     # 本文件
```

## 核心文件说明

### 1. outline.md - 学习大纲

**文件路径**: `outline.md`

**内容结构**:
- **第一部分：词法**（基础核心，单词的"用法规则"）
  - 第1-10章：名词、代词、冠词、数词、动词、形容词、副词、介词、连词、感叹词
  
- **第二部分：语法**（核心规则，句子的"搭建逻辑"）
  - 第1-8章：句子结构、时态、语态、语气、从句、非谓语动词、主谓一致、特殊句式
  - 每章包含多节课，每节课有知识点和练习方向
  
- **第三部分：词法与语法的综合应用**（衔接考点）
  - 第1-3章：词性转换、固定搭配与短语、常见错误辨析

**大纲特点**:
- 系统化：从词法到语法，从基础到进阶
- 分层设计：每章分为多节课，循序渐进
- 练习导向：每节课明确练习方向和重点

### 2. prompt/ - 提示词模板目录

**目录路径**: `prompt/`

**模板文件**:

#### template_mid.md（当前使用）
**适用场景**: 高中英语水平，逐步过渡到考研

**包含3套模板**:

**模板1：单小节基础版**
- 适用：每一节课
- 练习题：8-10题（单选5-6题、填空2题、造句2题）
- 口语练习：3个任务（跟读2句、情景问答1个）
- 难度：高中基础，少量中档题

**模板2：整章节综合版**（推荐用于生成章节练习）
- 适用：某一章（包含多节课）
- 练习题：12-15题（单选7-8题、改错3-4题、翻译2题）
- 口语练习：4个任务（跟读3句、情景描述2个）
- 附加：高频易错点汇总
- 难度：高中综合，铺垫考研

**模板3：高中→考研过渡版**
- 适用：需要逐步提升的场景
- 练习题：10题（标注"高中考点"/"考研铺垫考点"）
- 口语练习：3-4个任务（包含句式升级）
- 难度：循序渐进

#### template.md
**适用场景**: 考研英语导向
- 练习题更贴近考研题型（长难句拆解、翻译、写作）
- 口语练习更贴近考研口语
- 难度更高

### 3. src/ - 生成的练习题目录

**目录路径**: `src/`

**目录结构映射规则**:
```
src/
├── part1/           # 第一部分：词法
│   ├── chapter1/    # 第一章：名词
│   │   └── noun_exercises.md
│   ├── chapter2/    # 第二章：代词
│   │   └── pronoun_exercises.md
│   └── ...
├── part2/           # 第二部分：语法
│   ├── chapter1/    # 第一章：句子基本结构

│   │   ├── lesson2_exercises.md
│   │   └── lesson3_exercises.md
│   └── ...
└── part3/           # 第三部分：综合应用
```

**文件命名规则**:
- 词法章节：`{chapter_name}_exercises.md`（如 `noun_exercises.md`）
- 语法章节：`lesson{N}_exercises.md`（如 `lesson1_exercises.md`）
- 综合应用章节：根据内容命名

**练习题文件内容结构**:
```markdown
# 第X章：{章节名}

## 知识点概览
- 核心定义
- 分类
- 关键用法
- 易错点

## 综合练习题
### 一、单项选择题
### 二、改错题
### 三、翻译题

## 口语综合练习
### 任务一：核心句式跟读
### 任务二：简单情景描述

## 本章高中高频易错点汇总

## 学习建议
```

## 生成规则

### 规则1：目录创建
- 根据大纲的章节结构创建对应的目录
- 第一部分：`src/part1/chapter{N}/`
- 第二部分：`src/part2/chapter{N}/`
- 第三部分：`src/part3/chapter{N}/`

### 规则2：模板选择
- **词法章节**（第1-10章）：使用模板2（整章节综合版）
- **语法章节**（第1-8章，每章多节课）：使用模板2（整章节综合版）
- **综合应用章节**（第1-3章）：使用模板2（整章节综合版）

### 规则3：内容生成流程

**步骤1：提取大纲内容**
从 `outline.md` 中提取对应章节的完整内容，包括：
- 章节标题
- 所有知识点
- 所有练习方向

**步骤2：选择提示词模板**
根据章节类型选择合适的模板（推荐使用 `template_mid.md` 的模板2）

**步骤3：组合提示词**
```
【大纲内容】
---
【提示词】
```

**步骤4：生成练习题**
根据组合的提示词生成完整的练习题内容

**步骤5：保存文件**
按照命名规则保存到对应目录

### 规则4：难度控制
- 当前阶段：高中英语水平
- 练习题难度：基础为主，少量中档题
- 不考：考研难度的长难句和复杂题型
- 铺垫：悄悄铺垫考研相关的易错点

### 规则5：内容完整性
每个练习题文件必须包含：
1. 知识点概览（从大纲提取）
2. 综合练习题（12-15题）
3. 口语综合练习（4个任务）
4. 高频易错点汇总
5. 学习建议

## 使用流程

### 流程1：生成新的章节练习题

1. **确定章节**
   - 查看大纲，确定要生成的章节
   - 例如：第一部分第二章（代词）

2. **创建目录**
   ```
   mkdir -p src/part1/chapter2
   ```

3. **提取大纲内容**
   - 从 `outline.md` 复制对应章节的完整内容

4. **选择模板**
   - 打开 `prompt/template_mid.md`
   - 选择模板2（整章节综合版）

5. **组合提示词**
   - 将大纲内容粘贴到提示词前面
   - 使用分隔线 `---` 分隔

6. **生成练习题**
   - 将组合的提示词发送给大模型
   - 获取生成的练习题内容

7. **保存文件**
   - 按照命名规则保存到对应目录
   - 例如：`src/part1/chapter2/pronoun_exercises.md`

### 流程2：批量生成练习题

1. **规划生成顺序**
   - 建议按照大纲顺序逐章生成
   - 优先生成第一部分（词法），再生成第二部分（语法）

2. **使用模板2**
   - 所有章节都使用模板2（整章节综合版）
   - 保持难度一致性

3. **验证内容**
   - 检查练习题数量（12-15题）
   - 检查口语练习数量（4个任务）
   - 检查是否包含易错点汇总

### 流程3：更新已有练习题

1. **打开现有文件**
   - 找到对应的练习题文件

2. **对比大纲**
   - 检查是否覆盖所有知识点
   - 检查练习方向是否完整

3. **补充内容**
   - 如有遗漏，使用对应模板补充
   - 保持文件结构一致

## 注意事项

### 1. 大纲内容完整性
- 必须完整复制章节内容，包括所有知识点和练习方向
- 不要遗漏任何子知识点
- 保持markdown格式

### 2. 模板选择准确性
- 当前阶段统一使用 `template_mid.md`
- 优先使用模板2（整章节综合版）
- 如需调整难度，可在提示词末尾补充说明

### 3. 文件命名规范性
- 使用英文命名，避免中文
- 使用下划线分隔单词
- 保持命名一致性

### 4. 目录结构一致性
- 严格按照大纲结构创建目录
- 不要随意调整目录层级
- 保持映射关系清晰

### 5. 难度控制
- 当前阶段：高中英语水平
- 不考考研难度的长难句
- 重点巩固基础，铺垫考研

### 6. 内容质量
- 练习题必须附详细解析
- 口语练习必须可直接开口
- 易错点汇总必须简洁明了

## 常见问题

### Q1: 如何调整练习题难度？
在提示词末尾补充："适当调整练习题难度，贴合高中中等水平" 或 "适当增加难度，铺垫考研基础"

### Q2: 如何增加练习题数量？
在提示词中修改练习题数量要求，例如："综合练习题（15-20题）"

### Q3: 如何调整口语练习内容？
在提示词中修改口语练习的具体要求，例如增加或减少任务数量

### Q4: 生成的练习题不符合要求怎么办？
检查：
1. 大纲内容是否完整复制
2. 是否使用了正确的模板
3. 提示词是否组合正确
4. 可重新生成或手动调整

### Q5: 如何验证练习题质量？
检查：
1. 练习题数量是否符合要求
2. 是否覆盖所有知识点
3. 解析是否详细易懂
4. 口语练习是否可直接开口
5. 是否包含易错点汇总
