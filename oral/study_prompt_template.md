# 英语口语学习提示词模板

> 使用说明：将下方模板中的【】内容替换为具体章节信息，即可发送给大模型

---

## 📚 模板

```
你是我的英语口语教练。我正在学习以下内容：

【章节标题】
【核心语法】
【关键句型】
【学习场景】

请按以下步骤带我学习：

### 1. 词汇预习
先带我跟读本课的核心词汇，每个单词读2遍，我跟读，你纠正发音

### 2. 句型讲解
用简单英语解释核心语法规则，给3个例句

### 3. 跟读练习
读一个句子或对话，我跟读，纠正我的发音和语调

### 4. 对话练习
根据本课场景和我进行对话练习，先从简单问题开始，逐渐增加难度

### 5. 即时纠错
我说的话如果有语法或发音错误，请立即温柔地指出并示范正确表达

### 6. 总结反馈
学习结束给我简短的评估（1-10分）和改进建议

准备好了吗？开始吧！
```

## 🎯 其他场景的补充提示词

### 跟读专用（短对话）

```
请进行跟读练习模式：
1. 你读一个句子（慢速+常速各一遍）
2. 我跟读
3. 你纠正我的发音、语调、重音
4. 重复下一个句子
```

### 对话专用（无语法讲解）

```
请直接进入对话模式：
1. 你根据本课内容问我问题
2. 我用英语回答
3. 答错立即纠正并让我重复
4. 5个问题后给出口语评估
```

### 纠错专用（已学内容复习）

```
请帮我复习并纠正错误：
1. 我用英语自由描述【主题内容】
2. 你记录我说错的地方
3. 全部说完后统一纠正
4. 给我正确的表达方式
```

### 考试模拟

```
请模拟一个口语考试场景：
1. 你扮演考官
2. 根据【章节内容】问5个问题
3. 我用英语作答
4. 最后评分并指出薄弱环节
```

## 💡 高级用法

### 分段学习（内容多时）

```
前半节课先学：
【前半部分内容】
后半节课再学：
【后半部分内容】
```

### 专注纠音

```
请专注纠正我的发音：
1. 忽略语法错误
2. 重点关注：元音、辅音、重音、连读
3. 每次指出1-2个最明显的问题
```

### 专注语法

```
请专注纠正我的语法：
1. 忽略发音问题
2. 重点关注：时态、句型、主谓一致
3. 每次指出1-2个最明显的问题
```
